## Project Overview

Collectify is a static web application designed to help users organize and visualize their personal collections such as trading cards, vinyl records, and figurines in one unified platform.

Collectors often manage different types of items across multiple platforms, leading to a fragmented experience. This project addresses that problem by providing a clean and centralized interface where users can:

- Track collections across categories  
- Monitor purchase and estimated values  
- Visualize portfolio insights  
- Interact with a chatbot for quick queries  

The goal of this project is to build a structured and visually appealing prototype that can be extended into a full product.

## Botpress Integration

This project includes an AI-powered chatbot built using Botpress.

- A structured dataset of collectible items was created using Botpress Tables  
- The data was converted into a Knowledge Base  
- An Autonomous Node was configured for query handling  
- The chatbot is embedded across the website  

Example queries:
- “What is the estimated value of Black Lotus Alpha?”  
- “Which items belong to the Vinyl category?”  
- “Tell me about Pikachu Illustrator Promo”
